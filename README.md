## Spring Profiles

#### How to run using a profile?

`mvn spring-boot:run -DskipTests -Dspring-boot.run.profiles={Profile-Name}`

e.g. For Dev

```mvn spring-boot:run -DskipTests -Dspring-boot.run.profiles={Profile-Name}```
